1) threshold.py
Input the final wig.txt or mcaller_to_wig.txt.
Threshold is set as 0.5
Any site has the methylation score > 0.5, then the metylation score becomes 1.0
Any site has the methylation score <= 0.5, then the metylation score becomes 0

2) threshold_1.py
Input the final wig.txt or mcaller_to_wig.txt.
Threshold is set as 0.7
Any site has the methylation score > 0.7, then the metylation score does not change
Any site has the methylation score <= 0.7, then the metylation score becomes 0

3)noise_extinctiton_1.py
Input the file generated by threshold_1.py
The default size is 2kb
